This repository contains the work for Lab 2 of DSCI 560.

The lab focuses on collecting and exploring domain-specific data for a group chat AI assistant in the fitness domain. Public datasets were collected from multiple sources and processed into a unified format to support future chatbot development.

The lab includes:
- Collecting and exploring exercise data (CSV), online forum discussions (Reddit), and scientific articles (PDF/API)
- Converting heterogeneous data sources into structured CSV files
- Integrating all collected data into a single dataset for analysis

Libraries used: Python, pandas, requests, BeautifulSoup, pdfplumber.
